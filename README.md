=========================

Dash Documentation Lookup

[Dash](http://kapeli.com/) comes with default Alfred 2 Workflow. This is an extension to that by shortening the keyword filters for each language. Does not require online connection.

**Requires:** [Dash](http://kapeli.com/dash) >= 1.9.2

**Commands Included:** `dash {query}` (default), `html {query}`, `css {query}`, `js {query}`, `jquery {query}`, `jqueryui {query}`, `angularjs {query}`, `bootstrap {query}`, `svg {query}`, `nodejs {query}`, `php {query}`, `redis {query}`, `mysql {query}`, `man {query}`, `cpp {query}`, `sass {query}`, `compass {query}`, `backbone {query}`, `underscore {query}`,  `wordpress {query}`, `drupal {query}`, `django {query}`, `python {query}`, `python2 {query}`, `elisp {query}`, `android {query}`, `yui {query}`, `ruby {query}`, `rails {query}`, `gem {query}`, `bourbon {query}`, `neat {query}`, `jade {query}`, `stylus {query}`, `coffee {query}`, `knockout {query}`, `erlang {query}`, `lua {query}`, `foundation {query}`, `ios {query}`, `lodash {query}`
